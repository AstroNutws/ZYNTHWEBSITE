Schema design that feels visual, fast, and production-ready.

Zynth is a desktop schema designer for PostgreSQL and Supabase that helps teams move from rough ideas to real database structures without losing accuracy. You can model tables and relationships visually, edit constraints from a focused properties panel, and generate SQL or Prisma code in real-time.

The goal is simple: reduce friction between planning, implementation, and deployment so you can ship faster with fewer schema mistakes.

Who this is for

  • Developers learning relational modeling through visual feedback
  • Startup teams rapidly iterating product schemas
  • Backend engineers who need accurate SQL/Prisma output
  • Supabase users who want import + apply workflows in one desktop tool
  • Teams that review schema design before migration rollout

What Zynth is all about

Most schema workflows are split across whiteboards, SQL files, and DB consoles. Zynth unifies those pieces into one interface: visual modeling, detail editing, and generation pipelines all connected in a single desktop app.

From idea to structure

Start from a blank schema or imported database, add tables with non-overlap placement, define keys and constraints, then see relationships and dependencies clearly. This reduces ambiguity early and helps prevent migration rework.

From structure to deployment

Generate SQL/Prisma output live as you model. Optionally connect to PostgreSQL or Supabase to import existing schema state, scope by schema, and apply generated SQL with deliberate control.

Core features

Visual canvas

Drag tables, inspect relationship flow, and keep diagrams readable with spacing-aware placement and centering tools.

Rich column modeling

Configure datatype, PK, unique, nullable, default, ENUM values, and foreign-key target from the properties panel.

Realtime generation

Keep SQL, Prisma, and migration diff views synced to your current schema so code output is always up-to-date.

DB-connected workflow

Connect to PostgreSQL/Supabase, import a schema, and apply generated statements without leaving the app.

Team iteration

Use save/load with `.zynth` project files for handoff, review cycles, and reproducible schema planning sessions.

Keyboard + navigation support

Undo/redo, reset view, center selected table, and efficient pan/zoom controls to keep editing flow fast.
